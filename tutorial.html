Anaconda Installation Summary Guide

For Ubuntu

  1. $sudo apt-get install libgl1-mesa-glx libegl1-mesa libxrandr2 libxrandr2 libxss1 libxcursor1 libxcomposite1 libasound2 libxi6 libxtst6
  2. Download anaconda for linux here and put it in a directory of your choice.
  3. Verify data integrity of the downloaded file using SHA-256 $sha256sum path/to/downloaded_file/filename
  4. To install anaconda for python 3.7 $bash ~/path/to/downloaded_file/filename.sh or Anaconda3-2021.05-Linux-x86_64().sh
  5. Close and open your terminal window for the installation to take effect, or you can enter the command $source ~/.bashrc
  6. After filling the prompt from the terminal, use the code below to activate the conda environment $source <path to conda>/bin/activate
  7. Then run $conda init
  8. Finally, $anaconda-navigator

NOTE

Step 6 and Step 8 are important for using anaconda after a successful installation in ubuntu

  • Step 6: $source <path to conda>/bin/activate
  • Step 8: $anaconda-navigator or $ $jupyter-notebook
